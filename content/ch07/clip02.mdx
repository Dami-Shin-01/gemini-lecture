# NotebookLM으로 활동 자료 관리하기

JB 활동 자료가 쌓일수록 필요한 정보를 찾기 어려워집니다. NotebookLM으로 **JB 전용 지식 베이스**를 만들어봅니다.

## 왜 NotebookLM인가?

- 업로드한 문서에서만 답변 → 정확도 높음
- 출처 인용 → 어떤 문서의 어느 부분인지 바로 확인
- 여러 문서를 동시에 분석 → 패턴 발견 가능
