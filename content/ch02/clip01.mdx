# 간담회 기획안 & 진행 슬라이드 만들기

구성원의 목소리를 제대로 수집하려면, 간담회 자체가 잘 설계되어야 합니다. Gemini로 기획안과 진행 슬라이드 초안을 빠르게 만들어봅시다.

- 대상: 팀원 20~30명

- 형식: 소그룹 토론 + 전체 공유

[결과물 형식]

2. 진행 타임라인 (시간대별 활동)
3. 소그룹 토론 주제 3가지
4. 진행자 유의사항`}
</PromptBlock>

## 실습 2: 진행 슬라이드 구성

<PromptBlock title="슬라이드 구성 프롬프트">
{`위 간담회 기획안을 바탕으로 진행용 슬라이드 목차를 만들어줘.

- 슬라이드 수: 10장 이내
- 각 슬라이드에 제목 + 핵심 내용 2~3줄
- 참석자들이 보기 편한 구성으로
- 마지막 슬라이드에는 오늘 이야기를 마무리하는 메시지 포함`}
</PromptBlock>

## 실습 3: Google Slides로 제작 요청

슬라이드 목차가 완성되면, Gemini에게 **Google Slides에 바로 붙여넣을 수 있는 형태**로 정리해달라고 요청합니다. 더 나아가 Apps Script 코드를 생성해 자동으로 슬라이드를 만들 수도 있습니다.

### 방법 A — 슬라이드별 내용 정리 (바로 붙여넣기용)

<PromptBlock title="Google Slides 제작용 정리 요청">
{`위 슬라이드 목차를 Google Slides에 바로 적용할 수 있도록 정리해줘.

각 슬라이드마다 아래 형식으로 작성해줘:
[슬라이드 N]
제목: 
본문 내용 (불릿 2~3개):
발표자 노트 (진행자가 말할 내용 1~2문장):

전체 슬라이드 수는 10장 이내로 유지해줘.`}
</PromptBlock>

### 방법 B — Apps Script로 슬라이드 자동 생성

Gemini에게 Google Apps Script 코드를 요청하면, 슬라이드를 자동으로 생성할 수 있습니다.

<PromptBlock title="Apps Script 코드 생성 요청">
{`위 슬라이드 내용을 Google Slides로 자동 생성하는 Google Apps Script 코드를 작성해줘.

조건:
- 슬라이드 제목과 본문 내용을 자동으로 채워넣기
- 배경색은 흰색, 제목 폰트 크기 32pt, 본문 18pt
- 코드 실행 방법도 함께 설명해줘`}
</PromptBlock>

  steps={[

<HighlightBox type="tip" title="어떤 방법을 쓸까?">
  처음이라면 **방법 A**로 시작하세요. Gemini가 정리한 내용을 Google Slides에 직접 복사하면 됩니다.
  익숙해지면 **방법 B**로 자동화해보세요. 다음 번 간담회부터는 훨씬 빠르게 준비할 수 있습니다.
</HighlightBox>
