# Before/After 프롬프트 비교

같은 목적이라도 **프롬프트를 어떻게 쓰느냐**에 따라 결과가 완전히 달라집니다. 실제 업무 상황별 Before/After를 비교해봅시다.

## 1. 이메일 작성

<BeforeAfter
  before={{
    label: "Before",

  }}
  after={{
    label: "After",
    prompt: "우리 회사는 식품 유통업체야. 신규 거래처 A마트 구매팀장에게 첫 거래 제안 이메일을 써줘. 주력 상품은 프리미엄 간편식이고, 초도 물량 20% 할인을 제안하려고 해. 공식적이지만 친근한 톤으로, 300자 이내.",

  }}
/>

  before={{

## 3. 데이터 분석

<BeforeAfter
  before={{
    label: "Before",
    prompt: "이 데이터 분석해줘",
    result: "기본적인 통계 요약만 제공",
  }}
  after={{
    label: "After",
    prompt: "첨부한 엑셀은 우리 팀 6개월간 고객 문의 데이터야. 1) 월별 문의량 추이, 2) 문의 유형 TOP 5, 3) 평균 처리 시간 변화를 분석하고, 개선 제안 3가지를 포함해서 팀장 보고용으로 정리해줘.",
    result: "추이 분석, 유형 분류, 처리 시간 분석 + 구체적인 개선안까지 포함",
  }}
/>

<HighlightBox type="expert" title="Before/After의 차이점">
  좋은 프롬프트의 공통점은 명확합니다:
  - **누가** 쓸 건지 (대상/역할)
  - **무엇을** 원하는지 (구체적 지시)
  - **어떤 형태**로 (출력 형식)
  - **어떤 조건**에서 (제약사항)
</HighlightBox>

<HighlightBox type="tip" title="실습 과제">
  본인의 최근 업무 중 하나를 골라, Before/After 프롬프트를 직접 만들어보세요. 
  "내가 평소에 쓰던 방식"이 Before, "대화 캔버스를 적용한 방식"이 After입니다.
</HighlightBox>
