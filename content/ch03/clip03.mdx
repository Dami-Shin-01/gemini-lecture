# 나만의 커뮤니케이션 Gem 만들기

소통 가이드북을 Gem에 담아두면, 매번 긴 프롬프트 없이 클릭 한 번으로 꺼내 쓸 수 있습니다.

## Gems 기본 개념

**Gems = 미리 설정된 AI 비서**

- 역할, 규칙, 형식을 한 번만 설정
- 이후엔 상황만 입력하면 일관된 결과 제공
- 팀원과 공유 가능

3. 경영진에게 현장 의견 보고
4. 이견이 있을 때 의사 표명

[출력 형식]
- 메시지 본문 (슬랙용/이메일용 구분)

- 상대방 반응에 따른 후속 대응 힌트
