## 챕터별 핵심 프롬프트

<PromptBlock title="ch01 — Gemini 기본 대화: 업무 이메일 작성">
{`너는 [직급/직무] 전문가야.
[수신자]에게 보내는 [주제] 관련 이메일을 작성해줘.
[배경] [상황 설명 1~2문장]
[포함할 내용] [핵심 메시지 1] / [핵심 메시지 2] / [요청 사항]
[조건] 톤: [공식적/친근한/간결한] / 분량: [200자 이내] / 마무리: [회신 요청]`}
</PromptBlock>

<PromptBlock title="ch02 — NotebookLM: 문서 기반 분석 질문">
{`업로드한 자료를 바탕으로 다음을 분석해줘.
1. [문서 전체]에서 반복 언급되는 핵심 주제 [3~5]개를 뽑아줘
2. 각 주제별 대표적인 원문 인용구 1개씩 포함
3. 주제 간 연결 관계가 있다면 설명
4. [간담회/보고서]에 활용할 인사이트 3가지 제안
표 형식으로 정리해줘.`}
</PromptBlock>

<PromptBlock title="ch03 — Gem: 반복 분석용 인스트럭션">
{`너는 [역할명] 전문가야.
[역할] [핵심 역할 설명 1] / [핵심 역할 설명 2]
[분류 기준] 사용자가 [입력 데이터]를 입력하면:
1. [기준 1] — [설명] / 2. [기준 2] — [설명] / 3. [기준 3] — [설명]
[출력 형식] | 번호 | 원문 요약 | 분류 | [추가 열] |
[규칙] 원문 왜곡 금지 / 분류 애매 시 복수 표시`}
</PromptBlock>

<PromptBlock title="ch04 — Deep Research: 벤치마킹 조사">
{`[주제/산업]에 대해 심층 조사해줘.
[조사 범위] [경쟁사/선진사례] [3~5]개 기업의 [특정 영역] 현황 / 최근 [1~2]년 트렌드 / [우리 조직] 적용 시사점
[출력 형식] 1. 요약(300자) 2. 기업별 비교표 3. 핵심 인사이트 3가지 4. 추천 액션
출처를 반드시 포함해줘.`}
</PromptBlock>

<PromptBlock title="ch05 — Sheets =GEMINI(): 설문 응답 분류">
{`=GEMINI("다음 자유 서술 응답을 읽고, [카테고리1/카테고리2/카테고리3/기타] 중 가장 적합한 분류 1개만 답해. 부가 설명 없이 분류명만 출력해: "&[셀참조])`}
</PromptBlock>

<PromptBlock title="ch06 — Slides: 경영진 보고 슬라이드 구성">
{`너는 경영진 보고 전문 컨설턴트야. [5]장짜리 보고 슬라이드 구성안을 만들어줘.
[보고 내용] [핵심 메시지 1] / [핵심 메시지 2] / [데이터/수치]
[슬라이드 구성] 1장: 제목+요약 / 2장: 현황 / 3장: 분석 결과(차트) / 4장: 제안 / 5장: 실행 계획+타임라인
[조건] 경영진이 3분 안에 파악 / 슬라이드당 핵심 메시지 1개 / 구체적 수치 포함`}
</PromptBlock>

<PromptBlock title="ch07 — Canvas: 대시보드 웹앱 생성">
{`[주제] 대시보드 웹앱을 만들어줘.
[데이터] [표 형식으로 데이터 입력]
[구성 요소] [차트 유형 1]: [데이터] / [차트 유형 2]: [데이터] / 필터: [기준] / 색상: [규칙]
[요구사항] 반응형 레이아웃 / 호버 시 상세 정보 / [추가 인터랙션]`}
</PromptBlock>
