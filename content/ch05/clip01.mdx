# 캠페인 기획서 작성 실습

간담회에서 나온 결과를 실행으로 연결하려면, 구성원이 실제로 움직이게 하는 **캠페인**이 필요합니다. Gemini로 캠페인 기획서를 빠르게 만들어봅니다.

## 실습: 캠페인 기획서 작성

<PromptBlock title="캠페인 기획서 프롬프트">
{`너는 사내 문화 캠페인 전문 기획자야.

[작업]
JB 간담회 결과를 실행으로 연결하는 구성원 참여 캠페인 기획서를 작성해줘.

[캠페인 배경]
- JB가 구성원 간담회를 통해 주요 개선사항을 도출함
- 경영진이 3가지 사항을 실제 반영하기로 결정
- 이 내용을 구성원들에게 알리고 변화를 체감시키는 것이 목적

[기획서 구성]
1. 캠페인 개요 (목적, 대상, 기간)
2. 핵심 메시지 (슬로건 포함)
3. 단계별 실행 계획 (3주 기준)
4. 채널별 콘텐츠 계획 (사내 게시판 / 슬랙 / 오프라인)
5. 성공 지표 (KPI 2가지)

[조건]
- 예산 최소화 (디자인 툴, 외주 없이 진행)
- 분량: A4 2페이지`}
</PromptBlock>

## 실습 심화: 메시지 버전 비교

<PromptBlock title="캠페인 슬로건 만들기">
{`이 캠페인의 슬로건 후보를 5개 만들어줘.

조건:
- 구성원이 바로 이해할 수 있는 표현
- 긍정적이고 행동을 유발하는 톤
- 짧고 기억하기 쉬운 (10자 이내)

각 슬로건에 간단한 설명도 붙여줘.`}
</PromptBlock>
