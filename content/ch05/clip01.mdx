# 간담회 메모 → 액션 아이템

> ⏱ 이 챕터(ch05): 약 35분 | 이 클립: 약 12분

오후 1시 30분. 간담회가 끝났습니다. 김지연 대리의 노트북에는 손으로 끄적인 메모와 녹음 파일이 남아 있습니다. "참석자들이 정말 많은 이야기를 해줬는데... 이걸 30분 안에 실행 계획으로 정리해야 해." 점심 커피를 한 모금 마시고 Gemini를 엽니다.

## 회의 끝 = 실행 시작

간담회에서 나온 의견은 **48시간 안에 실행 계획으로 전환**하지 않으면 흐지부지됩니다. Gemini를 활용하면 회의록에서 바로 액션 아이템을 추출할 수 있습니다.

## 회의록 → 액션 아이템 변환 프롬프트

<PromptBlock title="회의록 → 액션 아이템 표 변환">
{`너는 프로젝트 매니저야. 아래 간담회 회의록을 분석해서 실행 과제 표로 변환해줘.

[회의록]
(여기에 간담회 메모/녹취록을 붙여넣으세요)

예시:
"현장에서 안전장비 교체 요청이 많았음. 특히 여름철 마스크가 너무 답답하다는 의견. 
신입사원 교육이 너무 형식적이라 실무에 도움이 안 된다는 불만.
팀장들은 회의가 너무 많아서 실무 시간이 부족하다고 함.
복지포인트 사용처 확대 요청 (현재 자사몰만 가능).
야근 시 석식 지원 기준이 불명확하다는 지적."

[결과물 형식]
| 번호 | 과제명 | 카테고리 | 담당 부서 | 기한 | 우선순위 | 비고 |
각 행에 대해:
- 과제명: 구체적인 실행 동사로 시작 (예: "~검토", "~개선", "~수립")
- 카테고리: 근무환경 / 제도개선 / 교육 / 복지 / 소통 중 선택
- 담당 부서: 예상 담당 부서 제안
- 기한: 즉시(1주) / 단기(1개월) / 중기(분기) 로 구분
- 우선순위: 상/중/하 (언급 빈도 + 영향 범위 기준)
- 비고: 관련 발언 요약 또는 유의사항`}
</PromptBlock>

## 우선순위 자동 정렬

액션 아이템이 너무 많으면 우선순위를 Gemini에게 맡길 수 있습니다.

<PromptBlock title="우선순위 재정렬 프롬프트">
{`위 과제들을 아래 기준으로 우선순위를 다시 정렬해줘.

[우선순위 기준] (가중치 순)
1. 구성원 체감도: 바뀌면 바로 "달라졌다" 느끼는 정도 (40%)
2. 실행 난이도: 예산/인력 없이 빨리 할 수 있는 것 우선 (30%)
3. 경영진 관심도: 이번 분기 경영 방침과 연결되는 것 (20%)
4. 언급 빈도: 여러 명이 반복적으로 말한 것 (10%)

Quick Win (체감도 높고 + 실행 쉬운 것)을 별도로 표시해줘.`}
</PromptBlock>

<HighlightBox type="expert" title="강사 팁">
  **Quick Win을 먼저 실행하세요.** "안전장비 교체 검토"처럼 예산이 큰 과제는 시간이 걸리지만, "석식 지원 기준 공지"처럼 공지 하나로 해결되는 것은 당장 실행할 수 있습니다. 빠른 실행 → 구성원 체감 → "간담회 해봐야 소용없다"는 인식 전환. 이 선순환이 JB의 핵심입니다.
</HighlightBox>

## 실습: 내 회의록으로 해보기

실제 간담회 메모가 없다면, 위 예시 회의록을 그대로 사용해도 됩니다. 생성된 액션 아이템 표를 Google Sheets에 복사하면 **진행 상황 추적 시트**로 바로 활용할 수 있습니다.

<HighlightBox type="tip" title="내 업무에 적용하기">
  - 이 프롬프트를 Gem으로 저장해두면 **매번 간담회 후 클릭 한 번**으로 액션 아이템을 추출할 수 있습니다
  - "회의 끝 → Gemini에 붙여넣기 → 액션 아이템 표 → 담당자에게 공유"를 루틴으로 만드세요
  - 액션 아이템 표는 다음 간담회 때 **"지난번 과제 이행 현황"**으로 재활용할 수 있습니다
</HighlightBox>
