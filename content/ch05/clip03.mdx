# 실행 계획 챌린지

지금까지 배운 기획과 카피 작성 능력을 결합하여 **실제 JB 활동 실행 계획**을 직접 만들어봅니다.

## 챌린지 소개

<HighlightBox type="important" title="챌린지 목표">
  이번 챌린지는 Gemini를 활용해 **나의 JB 활동 중 실제 실행할 1가지 계획**을 완성하는 것입니다.
  완성된 계획은 발표하거나 팀원과 공유해보세요.
</HighlightBox>

## 챌린지 과제

**Step 1.** 아래 중 본인의 현재 JB 상황과 가장 가까운 시나리오를 선택하세요.

<ComparisonTable
  headers={["시나리오", "상황", "목표"]}
  rows={[
    { label: "A", cells: ["간담회 결과 발표 준비 중", "결과를 구성원에게 효과적으로 공유"] },
    { label: "B", cells: ["새로운 JB 활동 제안 준비 중", "경영진의 승인을 받아야 함"] },
    { label: "C", cells: ["구성원 참여율이 낮아 고민 중", "참여 동기 부여 캠페인 필요"] }
  ]}
/>

**Step 2.** 선택한 시나리오에 맞는 실행 계획을 Gemini와 함께 완성하세요.

포함할 내용:
- 목적 & 배경 (2줄)
- 구체적 실행 3단계 (각 단계에 담당자, 기간, 결과물 포함)
- 예상되는 장애물 1가지 & 대응 방안
- 성공 기준 (어떻게 되면 잘 된 건지)

**Step 3.** 완성된 계획을 1분 안에 설명하는 엘리베이터 피치를 작성하세요.

<PromptBlock title="엘리베이터 피치 작성 요청">
{`위 실행 계획을 경영진에게 1분 안에 설명하는 엘리베이터 피치 스크립트를 써줘.

- 문제 → 해결 방안 → 기대 효과 → 요청사항 구조
- 데이터나 수치가 있으면 포함
- 자신감 있고 간결한 톤`}
</PromptBlock>

<HighlightBox type="tip" title="챌린지 결과 공유">
  작성한 실행 계획을 팀원과 공유하고 피드백을 받아보세요.
  다음 챕터 '설득하기'에서는 이 계획을 경영진에게 어떻게 보고할지 배웁니다.
</HighlightBox>
