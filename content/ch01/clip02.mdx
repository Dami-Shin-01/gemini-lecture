# 검색 vs AI: 무엇이 다른가

검색엔진과 AI는 모두 "질문에 답하는 도구"처럼 보이지만, 작동 방식과 결과물이 근본적으로 다릅니다.

## 언제 무엇을 써야 할까?

**검색이 더 나은 경우:**
- 최신 뉴스나 실시간 정보가 필요할 때
- 특정 웹사이트를 찾고 싶을 때
- 공식 문서의 정확한 원문이 필요할 때

**AI가 더 나은 경우:**
- 정보를 종합하여 정리해야 할 때
- 초안을 빠르게 만들어야 할 때
- 아이디어를 브레인스토밍하고 싶을 때
- 복잡한 개념을 쉽게 설명받고 싶을 때

<HighlightBox type="tip">
  가장 좋은 전략은 **둘 다 쓰는 것**입니다. 
  AI로 초안을 만들고, 검색으로 팩트를 확인하는 조합이 가장 강력합니다.
</HighlightBox>
