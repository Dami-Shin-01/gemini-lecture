# 엑셀 데이터 분석

Gemini에 엑셀 파일을 업로드하면 데이터 구조 파악부터 인사이트 도출까지 한 번에 처리할 수 있습니다.

## 실습: 파일 업로드 → 분석

<StepByStep
  steps={[

<HighlightBox type="tip" title="분석 전 필수 단계">
  분석 전에 먼저 **"이 데이터의 구조를 설명해줘"**라고 요청하세요. Gemini가 데이터를 정확히 이해한 상태에서 분석해야 결과가 정확합니다.
</HighlightBox>

## 프롬프트 예시

<PromptBlock title="매출 데이터 분석">
이 엑셀 파일의 매출 데이터를 분석해줘.

- 월별 매출 추이와 전월 대비 성장률
- 제품군별 매출 비중 (상위 5개)
- 매출이 급증/급감한 시점과 가능한 원인 추정
- 결과는 표 형태로 정리해줘
</PromptBlock>

<PromptBlock title="인사 데이터 분석">
이 직원 데이터를 분석해줘.

- 부서별 인원 현황과 평균 근속연수
- 최근 1년 퇴사율이 높은 부서와 패턴
- 연봉 분포 (사분위수)
- 인사 담당자가 주목해야 할 이상 신호가 있다면 알려줘
</PromptBlock>

<HighlightBox type="warning">
  엑셀 파일에 **개인정보**(주민번호, 연락처 등)가 포함되어 있다면, 해당 열을 삭제하거나 마스킹한 후 업로드하세요.
</HighlightBox>
