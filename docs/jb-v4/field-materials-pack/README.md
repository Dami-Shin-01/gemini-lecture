# JB Field Materials Pack (v1)

> 실습 1(듣기) · 실습 3(축적하기)에서 NotebookLM 소스로 활용하는 사전 구성 파일 팩.
> 모든 인물·부서·사건은 **가상**이며 교육용 샘플입니다.

## 구성

| 파일 | 용도 | 포맷 |
|---|---|---|
| `간담회_속기록_1Q_3건.md` | VoE 수집 기초 자료 (간담회 3회 분량) | Markdown |
| `익명_제보_샘플_10건.md` | 사내 익명 제보 채널 수신 자료 | Markdown |
| `구성원_설문_원본.csv` | 분기 몰입도 설문 응답 50건 | CSV |
| `경영진_답변_사례.md` | 과거 VoE에 대한 경영진 응답 이력 | Markdown |
| `이슈_CSV_샘플.csv` | M4 실행하기용 이슈 리스트 (임팩트/난이도 평가 대상) | CSV |

## 강사 운영 안내

1. 강의 시작 전 NotebookLM에 **공유 노트북** 1개 생성 → 5개 파일 모두 업로드
2. 공유 링크를 슬랙/채팅 등으로 수강생에게 배포
3. 수강생은 링크 클릭 → "복제"로 본인 계정에 노트북 생성
4. 이후 실습은 수강생의 개인 노트북에서 진행 (원본 오염 방지)

## 샘플 세팅의 설계 의도

- **다양한 감정 톤** (분노·체념·기대·회의·우려)이 고르게 분포 → 실습 3에서 "감정 강도 축 재분류" 질문이 의미를 가짐
- **3개 소스 간 교차 등장 이슈** 의도적 삽입 → 출처 추적(Grounding) 실습에서 "여러 소스에서 반복되는 이슈" 경험
- **모호한 표현·줄임말·업계 용어** 일부 포함 → RAG의 의미 기반 검색 강점 체감
